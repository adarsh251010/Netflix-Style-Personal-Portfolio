# Netflix-Style Personal Portfolio

A Netflix-inspired developer portfolio built to showcase experience, projects, skills, and interactive engineering modules in a cinematic, professional layout.

## About
This portfolio presents a structured, hiring-focused story of work and capability. It combines a cinematic home experience with dedicated detail pages and interactive modules that highlight engineering decision-making, state management, and UX discipline.

## Key Highlights
- **Cinematic Home Experience** with smooth section flow and consistent visual hierarchy
- **Experience & Projects Pages** with professional summaries and impact framing
- **Skills Detail Pages** with dedicated hero + tags for each skill area
- **Engineering Playground** featuring interactive modules (AI logic, state, performance)
- **Auto Tour Mode** that guides viewers through the portfolio end-to-end
- **Contact Page** with reveal-style details and a professional completion message

## Sections
- **Experience**: Role-based timeline with responsibilities and impact
- **Projects**: Selected builds with focus areas and repo links
- **Skills**: 8 dedicated skill categories with detail pages
- **Engineering Playground**: Interactive modules for logic, state, and performance
- **Contact**: Clean, professional contact presentation

## Tech Stack
- **Frontend**: React (Create React App)
- **Styling**: Custom CSS (Netflix-inspired)
- **Routing**: React Router

## Local Setup
```bash
npm install
npm start
```

## Build
```bash
npm run build
```

## Notes
- All visuals and interactions are crafted for a hiring audience.
- The Auto Tour can be interrupted by manual input to return control to the user.
